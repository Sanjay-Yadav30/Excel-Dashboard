# 📊 Orders Dashboard in Microsoft Excel

## 📌 Project Overview

This project presents an interactive **Orders Dashboard** built using **Microsoft Excel** to analyze pizza bun sales data. The dashboard provides valuable insights into product performance, daily sales trends, revenue generation, and customer order channels.

Using Pivot Tables, Pivot Charts, Slicers, and Excel formulas, the dashboard enables users to interactively explore sales data and monitor key business metrics for better decision-making.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Conditional Formatting
- Data Analysis

---

## ✨ Key Features

- Interactive Order Type filter (Online & Physical Visit)
- Product-wise sales analysis
- Product-wise revenue analysis
- Daily sales trend visualization
- Daily revenue trend analysis
- Dynamic KPI cards
- Interactive and user-friendly dashboard

---

## 📊 Dashboard KPIs

- Total Orders
- Total Revenue
- Average Revenue
- Average Discount

---

## 📂 Dataset

The dashboard is built using an orders dataset containing the following fields:

- Order ID
- Product ID
- Sale Date
- Product Name
- Order Type (Online / Physical Visit)
- Product Price
- Salesperson Name
- Quantity Sold
- Discount (%)
- Revenue
